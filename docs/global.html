AboutTextSliders()

Ce composant affiche une liste de composants TextSlider à partir d'un tableau de données importées. Il utilise la fonction map() pour parcourir le tableau et créer un composant TextSlider pour chaque entrée. Il utilise également un React.Fragment pour contenir tous les composants TextSlider enfants sans ajouter de noeud supplémentaire à l'arbre de rendu.
